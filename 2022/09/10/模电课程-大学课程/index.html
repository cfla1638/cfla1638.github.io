绪论

  • 电压源

    理想的电压源内阻为0，输出电压U = E - IR0, R0 = 0，因此输出电压恒定不变。当内阻远小于外电阻时，可近似看作理想电压源。

  • 电流源

    理想电流源的内阻为无穷大，因此输出的电流I = IS - U / R ，恒定不变。

第十四章 半导体器件

二极管

  • 导通电压：

    硅管：0.5V

    锗管：0.1V

  • 正向压降

    硅管：0.6 ~ 0.8 V

    锗管：0.2 ~ 0.3 V

  • 主要参数

    最大整流电流 IOM : 长时间使用所允许通过的最大正向平均电流

    反向工作峰值电压 URWM : 保证二极管不被击穿而给出的反向峰值电压，一般是反向击穿电压的1/2或2/3。

    反向峰值电流 IRM ：在URWM 下的电流

  • 作用

    1. 整流
    2. 限伏/钳位
    3. 隔离：两个二极管并联，某个二极管的电压高，则它优先导通。因而电压低的那一个二极管被隔离了。

稳压二极管

  1. 使用稳压二极管时一定要有限流电阻R，不然二极管必定会被烧坏。
  2. 稳压二极管的工作原理：当电源电压升高，使得稳压二极管两侧电压有升高的趋势时，此时稳压二极管的电阻升高，电流增大，引起干路限流电阻的电流增加，限流电阻的电阻不变，因此限流电阻的分压增大，抵消了电源增加的电压。
  3. 在分析使用稳压二极管之前，假设二极管尚未反向导通，先判断二极管两侧的电压，只有二极管反向击穿时，才能有稳压的效果。

三极管

  • 放大区，IE = IC + IB , IC = β IB ，发射结正偏，集电结反偏。

  • 截止区，IC = 0, UCE = UCC ， 约等于断开，此时 IB = 0或以下

  • 饱和区，此时IB 和IC 关系很小，IC = UCC / RC , UCE = 0, 相当于短路

  • 如何判断三极管的工作状态？

    1. 求出 IC Maximun = (Vcc - UCE) / RC
    2. 求出 IB
    3. 若βIB > ICMaximun 饱和
    4. 否则处于放大区

  • 参数

    硅管 UBE = 0.6 ~ 0.7 V

    β 杠 ： 静态电流放大系数

    β：动态电流放大系数

    ICBO

    ICEO

    ICM 集电极允许的最大电流，超过它放大系数将会下降

    U（BR）CEO

  • 如果UCE 固定，那么随着IB 的增加，三极管的工作状态会慢慢从放大状态转为过饱和状态。

第十五章 基本放大电路

放大电路的输入电阻：输入电路的等效负载电阻

放大电路的输出电阻：输出电路的等效内阻

共发射极放大电路（固定偏置放大电路）

(名字来源：因为偏置电路是固定的)

输入: UI或者IB 或UBE

输出：UO (UCE 的交流分量 阻容耦合)

静态值是指

  • IB
  • IC
  • UBE ：一般是 0.6 V(硅管是0.6 ~ 0.7)
  • UCE

静态值的直接求解法：

  • IB ： (UCC - UBE ) / RB
  • IC ： βIB
  • UBE ：一般是 0.6 V(硅管是0.6 ~ 0.7)
  • UCE : UCC - RC IC

静态值得图解法：

  • 三极管的输出特性曲线是IC - UCE (y - x) 因此，只要求出IC = f(UCE ) 即可
  • 由UCE = UCC - RC IC 因此只要求出UCC 和 RC 即可

偏置电路是指产生偏流IB 的电路，偏置电阻是指限流IB的电阻

非线性失真

原因：静态工作点不合适 或 信号太大

分类

  • 截止失真 : 静态工作点位置太低，输入产生的IB 在负半周截止了，导致输出的IC也截止了（负半周），导致UCE 的正半周截止了
  • 饱和失真 ： 静态工作点位置太高，输出的IC 在正半周饱和，导致输出的UCE 在负半周失真。

什么可以影响静态工作点：

最直接的因素是IC，可以影响IC的因素都可以影响静态工作点

第十六章 集成运算放大器

理想运放

开环电压放大倍数 ∞

差模输入电阻 ∞

开环输出电阻 0

  • 工作在线性区

    虚断：由于差模输入电阻为∞，可以认为两个输入端的电流 i+ = i- = 0

    虚短：由于开环电压放大倍数 ∞，可以认为 u+ = u-

    • 虚地，如果由于虚短，如果两个输入端有一个接地，相当于另一个也接地

  • 工作在饱和区

    虚段：仍然存在

    虚短：此时工作在饱和区，不能满足在线性区的放大关系，因此，两个输入端的电压不必相等，虚短不存在

    u+ > u- : uo = + Uo(sat)

    u+ < u- : uo = - Uo(sat)

信号运算

  • 反向比例运算

    运放工作在线性区

    只要运放的开环放大倍数很高，比例运算的放大能力就只与外接电阻有关

    这个放大倍数就是闭环放大倍数

    Uo = -(RF / R1) U1

  • 正向比例运算

    电压从正向端输入

    放大比例也只与外接电阻有关

    Uo = (1 + RF / R1) U1

  • 加法运算

    Uo = -[(RF/R11)U11 + (RF/R12)U12 + (RF/R13)U13]

    Uo = -(RF / R1)(U11 + U12 + U13)

  • 减法运算

    Uo = (1 + RF / R1 ) (R3 / (R1 + R2)) U12 - RF / R1 U11

    当R1 = R2, R2 = RF

    Uo = RF / R1(U12 - U11)

    当 R1 = R2 = R2 = RF

    Uo = U12 - U11
