const е’Ң constexpr зҡ„иҜҰи§Је’ҢеҢәеҲ«

еҸ‘иЎЁдәҺ 2022-09-26 жӣҙж–°дәҺ 2024-11-21 еҲҶзұ»дәҺ зҗҗзўҺи®°еҪ• йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

constе’Ңconstexpr

constexpr

еёёйҮҸиЎЁиҫҫејҸпјҡжҳҜжҢҮеҖјдёҚдјҡж”№еҸҳдё”еңЁзј–иҜ‘жңҹй—ҙе°ұеҸҜд»Ҙи®Ўз®—еӨ„з»“жһңзҡ„иЎЁиҫҫејҸ

еёёйҮҸиЎЁиҫҫејҸжҳҜз”ұ

  • ж•°жҚ®зұ»еһӢ
  • ж•°жҚ®еҲқе§ӢеҖј

еҶіе®ҡзҡ„гҖӮ

int a = 1 a дёҚжҳҜдёҖдёӘеёёйҮҸиЎЁиҫҫејҸпјҢеӣ дёәе®ғзҡ„ж•°жҚ®зұ»еһӢдёҚжҳҜеёёйҮҸ

const int b = get_val() bдёҚжҳҜдёҖдёӘеёёйҮҸиЎЁиҫҫејҸпјҢеӣ дёәе®ғзҡ„еҲқе§ӢеҖјдёҚжҳҜеёёйҮҸ

з”ұдәҺдҪҝз”Ёconst дёҚиғҪзЎ®е®ҡдёҖдёӘиЎЁиҫҫејҸжҳҜдёҚжҳҜеёёйҮҸпјҢеӣ жӯӨC++11дҪҝз”Ёconstexpr дҝ®йҘ°з¬ҰпјҢеҸҜд»Ҙи®©зј–иҜ‘еҷЁеңЁзј–иҜ‘жңҹй—ҙжЈҖжҹҘдёҖдёӘиЎЁиҫҫејҸжҳҜдёҚжҳҜеёёйҮҸгҖӮ

дёӢйқўиҝҷдёӘзЁӢеәҸжҳҜдёҚжӯЈзЎ®зҡ„пјҢеӣ дёәaиҷҪз„¶иў«const дҝ®йҘ°пјҢдҪҶе®ғе№¶дёҚжҳҜеёёйҮҸиЎЁиҫҫејҸпјҢе®ғеј•з”ЁдәҶйқһеёёйҮҸиЎЁиҫҫејҸпјҢеӣ жӯӨеңЁзј–иҜ‘жңҹй—ҙе°ұеҸҜд»ҘзЎ®е®ҡе®ғдёҚжҳҜеёёйҮҸиЎЁиҫҫејҸ

иҖҢеҸӘиҰҒе°Ҷa = b ж”№дёә a = 1 зЁӢеәҸе°ұдёҚдјҡжҠҘй”ҷдәҶпјҢеӣ дёәжӯӨж—¶aзҡ„зЎ®жҳҜдёҖдёӘеёёйҮҸиЎЁиҫҫејҸ

constexpr еҮҪж•°

жҷ®йҖҡзҡ„еҮҪж•°дёҚжҳҜеёёйҮҸиЎЁиҫҫејҸпјҢдҪҶжҲ‘д»¬еҸҜд»Ҙе®ҡд№үconstexpr зұ»еһӢзҡ„еҮҪж•°пјҢе®ғеҸҜд»ҘдҪңдёәеёёйҮҸиЎЁиҫҫејҸпјҢдҪҶиҝҷдёӘеҮҪж•°еҝ…йЎ»з®ҖеҚ•еҲ°еҸҜд»ҘеңЁзј–иҜ‘жңҹй—ҙиў«зЎ®е®ҡ

constexpr жҢҮй’Ҳ & еј•з”Ё

constexpr жҢҮеҗ‘зҡ„дҪҚзҪ®еҝ…йЎ»жҳҜе…ЁеұҖеҸҳйҮҸпјҢйқҷжҖҒеҸҳйҮҸпјҢжҲ–иҖ…еӯ—йқўеҖјзӯүж”ҫеңЁ.bss иҠӮ.data дҪҚзҪ®зҡ„ж•°жҚ®пјҢиҖҢдёҚиғҪжҳҜж”ҫеңЁж ҲдёҠзҡ„жң¬ең°еҸҳйҮҸпјҢеӣ дёәд»–д»¬еҝ…йЎ»еңЁиҝҗиЎҢж—¶жүҚиғҪзЎ®е®ҡдҪҚзҪ®пјҢиҖҢдёҚиғҪеңЁзј–иҜ‘ж—¶зЎ®е®ҡдҪҚзҪ®гҖӮ

const

constз”ЁжқҘдҝ®йҘ°еҸҳйҮҸпјҢж №жҚ®const еҮәзҺ°зҡ„дҪҚзҪ®пјҢеҸҜд»ҘеҲҶдёәдёӨз§Қconst пјҡйЎ¶еұӮconst е’Ңеә•еұӮconst

йЎ¶еұӮconst еҸҜд»ҘиЎЁзӨәд»»ж„Ҹзҡ„еҜ№иұЎжҳҜеёёйҮҸ

дҪҺеұӮconst дёҺжҢҮй’Ҳе’Ңеј•з”Ёжңүе…іпјҢиЎЁзӨәпјҢдёҚиғҪйҖҡиҝҮиҝҷдёӘйҖ”еҫ„дҝ®ж”№д»–д»¬жҢҮеҗ‘жҲ–иҖ…еј•з”Ёзҡ„еҸҳйҮҸ

const дёҺеҸӮж•°дј йҖ’

  • еңЁеҸӮж•°дј йҖ’ж—¶пјҢйЎ¶еұӮconstдјҡиў«еҝҪз•ҘжҺүпјҢеӣ жӯӨдёӢйқўиҝҷдёӨдёӘеҮҪж•°дёҚиғҪйҮҚиҪҪпјҢеӣ дёәд»–д»¬еҝҪз•ҘжҺүйЎ¶еұӮconst еҗҺжІЎжңүеҢәеҲ«

  • еҸҜд»ҘдҪҝз”ЁйқһеёёйҮҸеҲқе§ӢеҢ–еә•еұӮconst пјҢдҪҶдёҚиғҪдҪҝз”ЁеёёйҮҸеҺ»еҲқе§ӢеҢ–йқһеә•еұӮconst

  • йқһеёёйҮҸзҡ„еј•з”Ёеҝ…йЎ»жҳҜе·ҰеҖјпјҢеёёйҮҸзҡ„еј•з”ЁеҸҜд»ҘжҳҜе·ҰеҖјпјҢд№ҹеҸҜд»ҘжҳҜеҸіеҖј

    еӣ жӯӨе°ҪйҮҸе°ҶдёҚдјҡж”№еҸҳеҪўеҸӮзҡ„еҮҪж•°зҡ„еҪўеҸӮе®ҡд№үжҲҗеә•еұӮconst пјҢеӣ дёәиҝҷж ·е®ҡд№үжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёеӯ—йқўйҮҸпјҲrvalueпјүгҖӮ
